# node_tdd

API REST in nodeJs for apply TDD concepts in backend using jest for unit test, and super test for functional tes

## Development server

Run `npm run start-dev` for a dev server. Navigate to `http://localhost:3000/`. The app will automatically reload if you change any of the source files.

## Dependecies

Run `npm install` for install all dependencies

## unit test and functional test

Run `npm run test:coverage` for running all test into server
